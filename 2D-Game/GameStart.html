Thread the Cube


Thread the Cube is a fast paced game that can not be put down.
The objective of the game is to get the highest score possible.

Instructions on how to play the game

To play the game is pretty simple, all you need to do is press the up arrow to go up, and press the down arrow to go down. There is a catch however. The catch is that the player will need to touch the ground in order to reset their boost, otherwise the player will be trapped on the ground and the game will end.
